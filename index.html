CSS Box Model

The box model comprises the set of properties that define parts of an element that take up space on a web page. The model includes the content area’s size (width and height) and the element’s padding, border, and margin.

How does the Box Model look like?

Box Model Properties Explained

Box Model Property What it does Semantic and logical Rules
width and height the width and the height of the content area write it in px, it will be responsive on all devices then
border line that surrounds an element, like a frame we specify it with width, style and color*
padding space between the contents of box and the borders of a box from all the sides, from 4 to 1 values or you can be specific
margin space directly outside the box, specify the size od that space from all the sides, from 4 to 1 values or you can be specific

*What we can specify border with

Border specification What it does What options do we have
width the thickness of the border border's thickness can be set in pixels or with one of the following keywords:thin, medium, or thick
style the design of the border web browsers can render any of 10 different styles. Some of these styles include:none, dotted, and solid
color the color of the border web browsers can render colors using a few different formats, including 140 built-in color keywords.
